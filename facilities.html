Laboratory Infrastructure

Laboratory Infrastructure

Our laboratory hosts advanced platforms for next-generation wireless research. Click on each system below to explore architecture, use cases, tools, and capabilities.

  • Architecture: End-to-end 5G core supporting both Standalone (SA) and Non-Standalone (NSA) modes.
  • Use Cases: Smart campus, private enterprise networks, network slicing experiments.
  • Tools Used: Open5GS, UERANSIM, srsRAN, custom core modules.
  • Capabilities: Testing and validating new 5G technologies, applications, and services before large-scale deployment.
  • Architecture: Deep Radio is an SDR based platform to process real-time signals.
  • Use Cases: Waveform design, spectrum sensing, Live FM transmitter.
  • Tools Used: GNU Radio, Python, USRP/LimeSDR, TensorFlow/PyTorch.
  • Capabilities: Real-time signal processing, deep-learning-based optimization.
  • Architecture: Wi‑Guy is an SDR‑based educational platform for learning wireless communication fundamentals.
  • Use Cases: Hands‑on SDR experiments, Academic and training usage, RF Spectrum Scanning
  • Tools Used: Wi-Guy hardware modules, custom RF configurations.
  • Capabilities: Decoding FM Radio Stations, Decoding 2G(GSM) Signals, channel analysis.
  • Architecture: The IoT Security Testbed enables experimentation on security, privacy, and resilience of IoT networks.
  • Use Cases: Smart campus, industrial automation, cyber-physical systems.
  • Tools Used: MQTT, Node-RED, Edge servers, custom APIs.
  • Capabilities: Device orchestration, data analytics, real-time monitoring.
